Concicional simples

Estruras condicionais servem para alterar o fluxo de execução do meu algoritimo Apos um teste de condição, se a condição for verdeira, o fluxo de execução do código interno à estrutural condicional executa. Caso contarario, o codigop salta para o final da estutura condicional.

if (teste de condição){ comandos executados quando a condição é verdadeira! }
